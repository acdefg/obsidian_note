# 截图软件 flameshot
## 安装

```txt
sudo apt install flameshot
```

## 添加快捷键
![设置截屏快捷键|600](https://raw.githubusercontent.com/acdefg/cdn/main/obsidian/20220415161137.png)

打开设置的 keyboard，名字随便输，command 输入
```shell
flamshot gui
```
