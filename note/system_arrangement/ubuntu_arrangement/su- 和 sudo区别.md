- `su -` : `swith users`
- `sudo` 的英文全称是 `super user do`
- 前者输入 `sudo su -` 后，需要提供当前用户的登录密码，也就是 ubuntu 用户的密码；
- 后者输入 `su -` 后，需要提供 root 用户的登录密码。
