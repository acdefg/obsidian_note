NeRF（神经辐射场）作为近年来**三维重建**领域最受瞩目的技术，其主要作用是通过输入一组从不同位置拍摄的静态照片，学习并生成该场景在任意未拍摄角度下的高质量图像。在核心原理上，NeRF 将复杂的三维世界建模为**一个连续的五维函数**，利用神经网络将空间坐标与观察视角映射为相应的颜色值和体积密度。渲染时，它通过相机射线对空间点进行采样，并运用体渲染技术将光线路径上的属性进行积分叠加，从而还原出逼真的视觉效果。这一算法的诞生具有深远的意义，它标志着三维表示**从传统的点云、体素或三角网格等“显式”结构向“隐式”神经表征的重大转变**，极大地提升了处理复杂光照、镜面反射及精细纹理的能力。

从技术演进的角度看，NeRF 的前身可以追溯到传统的多视图几何（MVS）和基于体素的场景重建方法，但它通过神经网络打破了物理分辨率与存储空间的限制。在其发展之后，涌现出了诸如 Instant NGP 等大幅提升训练与推理效率的优化方案，以及目前在工业界大放异彩的 3DGS（3D 高斯泼溅），后者通过将隐式场重新显式化为数百万个带有属性的高斯椭球体，解决了 NeRF 难以实时渲染的痛点。目前，NeRF 及其变体已被广泛应用于数字孪生、影视虚拟制片、增强现实内容创作以及自动驾驶仿真等领域。尽管它在处理超大规模场景和动态物体时仍面临挑战，但其对三维空间信息的紧凑表达和高保真重建效果，已成为现代计算机视觉研究不可或缺的基石

## 模型结构和输入输出
![image-16](https://imag060625.oss-cn-beijing.aliyuncs.com/img/20251218232857258.png)

NeRF 算法的核心在于将一个复杂的三维场景编码进一个多层感知机（MLP）的权重中，其输入包括空间点的三维坐标 $\mathbf{x} = (x, y, z)$ 以及观察该点时的单位方向向量 $\mathbf{d} = (\theta, \phi)$。由于神经网络在处理低维直接输入时难以学习到高频的细节特征，模型首先通过位置编码 $\gamma(p) = (\sin(2^0 \pi p), \cos(2^0 \pi p), \dots, \sin(2^{L-1} \pi p), \cos(2^{L-1} \pi p))$ 将这些输入映射到高维空间。在模型结构上，NeRF 设计了一个非对称的架构，位置 $\mathbf{x}$ 首先经过数层 MLP 输出该点的体积密度 $\sigma$ 和一个中间特征，随后这个特征才与视角 $\mathbf{d}$ 融合并经过最后的网络层输出颜色 $\mathbf{c} = (r, g, b)$。这种结构设计隐含了一个物理假设，即物体的几何形状（密度）是客观存在的，不随观察视角变化，而物体的外观颜色（如高光、反光）则会随着视角的变化而改变。

## 体渲染
在渲染阶段，NeRF借用了经典光学中的体渲染理论。对于从相机中心 $\mathbf{o}$ 出发、方向为 $\mathbf{d}$ 的射线 $\mathbf{r}(t) = \mathbf{o} + t\mathbf{d}$，我们关注其在近平面 $t_n$ 到远平面 $t_f$ 之间的颜色积累。该射线预测出的颜色 $C(\mathbf{r})$ 可以表示为积分形式

$$C(\mathbf{r}) = \int_{t_n}^{t_f} T(t) \sigma(\mathbf{r}(t)) \mathbf{c}(\mathbf{r}(t), \mathbf{d}) dt$$

这里的 $T(t)$ 被称为累积透射率，表示光线从 $t_n$ 传播到 $t$ 过程中未被吸收的概率。其推导基于简单的物理过程：假设光线在极短距离 $dt$ 内被吸收的概率正比于该点的密度 $\sigma(t)$，则透射率的变化遵循微分方程 $dT = -T(t)\sigma(t)dt$，求解该方程并设定边界条件 $T(t_n) = 1$，即可得到

$$T(t) = \exp\left(-\int_{t_n}^t \sigma(s) ds\right)$$

为了在计算机上实现，我们需要对上述积分进行离散化处理。通过将射线划分为 $N$ 个均匀区间并进行分段随机采样，我们得到离散渲染公式 $\hat{C}(\mathbf{r}) = \sum_{i=1}^N w_i \mathbf{c}_i$，其中每个采样点的权重为 $w_i = T_i (1 - \exp(-\sigma_i \delta_i))$，且 $T_i = \exp(-\sum_{j=1}^{i-1} \sigma_j \delta_j)$，这里的 $\delta_i$ 是相邻采样点之间的步长。

为了解决采样效率低下的问题，NeRF引入了层级化采样策略，即二次采样。系统同时维护两个网络：粗糙网络（Coarse）和精细网络（Fine）。首先在射线上进行均匀的分层采样得到一组点，通过粗糙网络计算出一组初始权重 $w_i$。这些权重实际上构成了一个关于射线长度 $t$ 的分段常数概率密度函数（PDF）。我们根据这个PDF进行逆变换采样，在权重较大的区域（即可能存在物体表面的位置）追加采样 $N_f$ 个点。最后，将所有的采样点一起输入精细网络进行渲染。这种做法能显著减少在空白区域的无效计算。在优化过程中，损失函数采用了非常直观的均方误差（MSE），即通过最小化渲染像素颜色与真实照片颜色之间的差距来训练模型

$$\mathcal{L} = \sum_{\mathbf{r} \in \mathcal{R}} \left[ \|\hat{C}_c(\mathbf{r}) - C(\mathbf{r})\|_2^2 + \|\hat{C}_f(\mathbf{r}) - C(\mathbf{r})\|_2^2 \right]$$

通过这种端到端的优化，模型最终能够将整张照片的信息“压缩”进网络参数中。
