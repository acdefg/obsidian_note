# 超参数

| 超参数                    | 作用                            |
| ---------------------- | ----------------------------- |
| **学习率（Learning Rate）** | 每次参数更新的步长大小                   |
| **Batch Size**         | 每步梯度计算使用的样本数                  |
| **学习率调度器**             | 学习率如何随时间变化（WSD、Cosine、Linear） |
| **Warmup Steps**       | 训练初期学习率从 0 线性升温到峰值的步数         |
| **Weight Decay**       | 参数正则化强度，防止过拟合                 |
| **梯度裁剪阈值**             | 限制梯度范数上限，防止爆炸                 |
